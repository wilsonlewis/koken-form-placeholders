## About

A <a href="http://koken.me">Koken</a> plugin that adds the label as placeholder text for built-in contact forms.

- <a href="http://koken.me">Koken</a> is a CMS focused on photography

## Installation

Koken does not have a store for 3rd party plugins, so we need to install it manually.

- <a href="http://koken.me/#dlkoken">Install Koken</a>
- Navigate to the plugins folder:

        cd /storage/plugins

- Git clone or download and unzip this package:
        
        git clone https://github.com/wilsonlewis/koken-form-placeholders.git
        
- Open a new browser window to login to your admin plugins section: 

        http://yoursite.com/admin/#/settings/plugins

- Click the **Enable** button next to this plugin to enable

## How to

Just add a Koken contact form to any <a href="http://help.koken.me/customer/portal/articles/632095-text-overview">essay or custom page</a>.

Then, you can hide the label with custom <a href="https://www.w3schools.com/css/css_display_visibility.asp">CSS</a> if you want.
